Real-Time Ball Balancing System

PID control and computer vision on a 3-servo platform

Python C++ MATLAB PID Control Computer Vision

A real-time control system that keeps a ball balanced at a target position on a tilting platform, driven by a PID controller and a computer-vision pipeline for position tracking.

The platform is actuated by three servos, giving it two degrees of tilt freedom. An overhead camera tracks the ball's position in real time, feeding coordinates into a PID loop that continuously adjusts platform angle to drive the ball toward a target point.

Highlights

• Designed and tuned a PID controller for two-axis tilt stabilization
• Implemented a computer vision pipeline for real-time position tracking of the ball
• Programmed across Python, C++, and MATLAB
• Controlled a 3-servo platform in real time
