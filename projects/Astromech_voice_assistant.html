Astromech Voice Assistant

An interactive voice assistant inspired by R2D2 Astromech Droid.

Voice Recognition Natural Language Processing Python APIs Raspberry Pi Electronics

A fully functional AI-powered voice assistant built inside a 22cm tall 3D-printed miniature astromech model, running on a Raspberry Pi 5. The project combines hardware electronics, mechanical design, and modern AI APIs to create a characterful desktop companion that responds to voice commands, displays animations, and reacts to its environment with lights and sound.

System Flowchart

• Listening for the wakeword "Hey R2"
• Play sound effects when wakeword is detected, start listening for command or question.
• Using Text-to-Speech (TTS) and sends it to Gemini API
• receives responses from Gemini API and sends it to ElevenLabs Voice Text to Speech API
• Plays the response audio through the speaker.
