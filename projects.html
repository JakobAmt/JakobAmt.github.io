Hobby Buisness and 3D modelling

Designed and manufactured custom 3D models to display different writstwatches with unique motifs.

Drag to rotate · scroll to zoom (sample model — swap the data-stl path for your own file)

ML Quadcopter

A quadcopter built to fly autonomously using an onboard machine learning model for object detection and obstacle avoidance. Covers the frame build, flight controller setup, and training the vision model.

ROS2 Robot using lidar to solve labyrinth with SLAM

A ros2 simulation of a Roomba robot using differential drive equipped with a lidar sensor for navigation and mapping. Driving autonomously through a small labyrinth exploring and mapping its environment. Demo video below doesnt show the SLAM mapping in action, only the lidar with different settings. This one uses a PD controller to avoid the walls

R2D2 Voice Assistant

A functional miniature R2D2 replica with a built-in voice assistant, running on a Raspberry Pi for movement and sound. Handles wake-word detection, voice commands, and classic R2D2-style beep responses.

Real-time Ball Balancing System

A ball-balancing platform combining two PID controllers with computer vision to track and stabilize a ball in real time. Built with a 3-servo platform, controlled through Python, C++, and MATLAB.

Acoustic Indoor Positioning System for Autonomous Drones (Bachelor Thesis)

Investigated ultrasonic time-difference-of-arrival (TDOA) for indoor drone localization, aimed at theatrical stage environments. Fixed anchor nodes triangulate the drone's position in real time, with the full pipeline built in Python and C++.
